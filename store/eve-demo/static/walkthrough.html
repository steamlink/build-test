Set up your first steamlink testnode!

1. Install Arduino software on your computer

  • Follow the instructions on https://www.arduino.cc/en/Main/Software

2. Install the RadioHead library

  • Download the RadioHead library and place it in your Arduino Libraries directory, which is in your Arduino folder
  • The latest version can be found at http://www.airspayce.com/mikem/arduino/RadioHead/RadioHead-1.65.zip
  • On Linux machines, the Arduino folder is named “Sketchbook” and it is typically located in /home/<username>
  • On Windows and Macintosh machines, the default name of the Arduino directory is “Arduino” and is located in your Documents folder.

3. Download the steamlink source code

  • Either Git Clone or download from: https://github.com/steamlink/build-test
  • Move or copy the SteamLink directory (which includes the Arduino Library) to your Arduino Libraries directory.

4. Open the SL_testclient0.ino file with the Arduino software

  • From the SL_testclient0 directory within build-test directory, open the SL_testclient0.ino file with the Arduino software

5. Add your node on the dashboard

  • Open https://demo.steamlink.net in your browser
  • Add a node with the correct mesh and swarm choice (you do not have to create a new mesh or swarm, there is a demo mesh and swarm created for this workshop)
  • Keep a note of your SLID and your node token. Your node token needs to be added to the top of your SL_testclient0.ino file like so:
    #define SL_TOKEN "YOUR_NODE_TOKEN_GOES_HERE"
